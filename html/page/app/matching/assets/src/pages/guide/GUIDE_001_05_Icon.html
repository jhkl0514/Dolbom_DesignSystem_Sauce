아이콘은 기능, 행동, 사물의 상징을 기호로 만든 시각 언어로 한정된 공간에서 효율적으로 정보를 전달하는 역할을 합니다.

1. 24x24

라인 형태의 디폴트 아이콘은 24x24사이즈를 사용하며, 상하좌우 2 여백을 제외한 공간에 배치합니다.

2. 40x40

이미지 형태의 아이콘 중 40x40사이즈는 상하좌우 4 여백을 제외한 공간에 배치합니다.

3. 80x80

이미지 형태의 아이콘 중 80x80사이즈는 여백 상관없이 해당 사이즈 안에 배치합니다.

4. 58x58

이미지 형태의 아이콘 중 58x58사이즈는 여백 상관없이 해당 사이즈 안에 배치합니다.
