Typography
타이포그래피(Typography)는 서비스와 사용자가 커뮤니케이션하는 주요 요소입니다. 텍스트 목적과 기능에 따라 서체의 스타일을 유의해 사용합니다.

Typeface

안드로이드, 웹 폰트는 국문/영문/숫자 모두 SUIT 폰트를 사용합니다. 타이포그래피 토큰 이름은 4가지 요소로 구성됩니다. Figma 및 CSS 라이브러리에서는 동일한 요소 이름이 공유되지만, 사용에는 약간의 차이가 있습니다. 
No. 범주 설명
1 폰트 사용한 폰트명을 노출합니다.
2 타입 제목, 본문, 캡션에 사용하는 폰트를 나타냅니다.
3 폰트 크기 알파벳 문자를 사용하여 인쇄 시스템의 상대적 크기를 나타냅니다.
XL, LG, MD, SM, SS, XS 사용됩니다.
4 폰트 두께 폰트 두께를 나타냅니다.
CSS 표기 : Regular(400), Semibold(700), Bold(800)
