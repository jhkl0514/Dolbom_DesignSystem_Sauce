Object Styles
객체 UI 시각적 일관성을 유지하기 위해 Radius, Border Width, Shadow의 세 가지 시각적 요소로 분류합니다.
버튼, 텍스트 입력 상자 등 디자인 구성 요소에 적용되는 스타일 속성을 정의합니다.

Radius

대부분의 구성요소에 Radius가 적용됩니다. Radius 값은 8의 배수를 사용하는 것을 권장합니다.

Key Radius, 핵심 반경 값

Raduis px 설명
radius-8 8 버튼, 폼형태의 컴포넌트 구성 요소에 사용되는 값
radius-16 16 팝업, 페이지,박스 형태에서 주로 사용되는 값
radius-40 40 바텀시트 페이지에서 사용되는 값
radius-circle 999 원형의 모양에서 사용되는 값

Table of Figma Propertie, 피그마 속성

spacing px spacing px
radius-2 2 radius-12 12
radius-4 4 radius-12 12
radius-4 4 radius-16 16
radius-6 6 radius-40 40
radius-8 8 radius-circle 999

Border Width

테두리는 3 가지(1px,1.5px, 2px) 형태를 제공합니다.
border-2(1.5px)은 페이지 단위에서는 사용을 지양하며, 아이콘에서만 사용할 것을 권장합니다.

Table of Figma Propertie, 피그마 속성

Raduis px 설명
border-1 1 가장 자주 사용되는 디폴트 값
border-2 1.5 아이콘에서만 사용할 것을 권장
border-3 2 강조가 필요한 영역에 사용하는 값

Elevation

연회색 바탕에 주로 사용되는 그림자는 3 가지 타입으로 정의합니다.

Table of Figma Propertie, 피그마 속성

shadow Value 설명
shadow-1 0px 2px 16px rgba (0, 0, 0, 0.08),
0px 0px 1px rgba (0, 0, 0, 0.2) 		페이지, 카드 형태에서 주로 사용하며, 가장 일반적인 그림자 값
shadow-2 0px 8px 16px rgba (0, 0, 0, 0.12),
0px 0px 8px rgba (0, 0, 0, 0.12) 		팝업에서 사용하는 그림자 값
shadow-3 0px -4px 16px rgba (0, 0, 0, 0.12) 바텀 내비게이션에 사용되는 그림자 값
shadow-4 0px 2px 4px rgba (0, 0, 0, 0.2) 하단영역 강조가 필요할 때 사용되는 그림자 값
shadow-5 0px 2px 16px rgba (0, 0, 0, 0.12),
0px 0px 1px rgba (0, 0, 0, 0.2) 		페이지, 카드 형태에서 사용하는 그림자 값(Shadow-1번보다 강조)
