Border Color
영역을 구분하는 테두리 색상입니다. 텍스트, 배경, 아이콘 색상과 중복되는 색상이 있어도 지정된 토큰 색상으로만 사용합니다.

구성요소 예시

Color/border/Primitive

브랜드 색상, 컴포넌트를 구성하는 색상입니다.
토큰 설명 참조값 Hex 컬러
Primary 활성화 시 사용되는 테두리 색상(예시: 버튼, 입력폼) brand/600 #FF8120
primary-press 활성화 시 사용되는 테두리 색상(강조) brand/800 #EE6002
secondary 컴포넌트 내 기본 테두리 색상 (예시 : 입력폼) neutral/300 #E0E0E0
secondary-press 활성화 시 사용하는 테두리 색상 neutral/500 #999999
inverse 진한 배경에 이미지나, 썸네일이있을 경우 사용하는 테두리 색상 Base/white #FFFFFF

Color/border

전체 화면에서 자유롭게 사용하는 색상입니다.
토큰 설명 참조값 Hex 컬러
Primary 첫번째 테두리 색상 neutral/400 #BDBDBD
secondary 두번째 테두리 색상 neutral/200 #EEEEEE
teritiary 세번째 테두리 색상 neutral/100 #F5F5F5
info 뱃지, 정보 등 신뢰 색상 Infomation/600 #1477F8
info-subtle 뱃지, 정보 등 신뢰 색상_서브 Infomation/100 #DBEAFE
success 완료 및 성공 색상(예시: 입력폼) success/600 #43A047
success -subtle 완료 및 성공 색상_서브 success/100 #C8E6C9
danger-bold 주의 및 에러 색상(예시: 입력폼) danger/800 #1E40AF
danger-subtle 주의 및 에러 색상_서브 danger/100 #FFCDD2
disabled 비활성화 색상 neutral/400 #BDBDBD
base 가장 진한 테두리 색상 base/black #111111
