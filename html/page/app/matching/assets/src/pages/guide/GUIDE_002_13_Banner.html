Banner
배너(Banner)는 특정 기능 또는 업데이트된 서비스, 광고, 프로모션 등 간략하고 중요한 메시지를 표시하는 안내 역할로 사용됩니다.

Anatomy, 컴포넌트 내부요소

  • 1. Container
  • 2. Title
  • 3. SubText
  • 4. Paging
  • 5. image
  • 6. category

Type, 종류

Simple
Extend

Spec, 제작가이드

1.Simple형 - 서비스 메인, 전체메뉴 노출
  • 사이즈 : 640x200(320x100, 2배수)
  • 텍스트: 타이틀, 설명 텍스트는 관리자에서 설정
- 타이틀 : Suit 18px Bold
- 설명 텍스트 : Suit 14px Regular
  • 이미지 : 이미지 영역만 PNG로 추출, 392x400추출(2배 사이즈)
  • 배너 노출 개수 : 최대 5개, 자동롤링(5초)
1-1.배경색상
배경색상은 neutral-200(#EEEEEE)고정입니다. 다른색상은 사용할 수 없습니다.
2. Extend형 - 길잡이 노출
  • 사이즈 : 640x480(320x240, 2배수) , Border radius :16px
  • 텍스트: 카테고리, 타이틀, 설명 텍스트는 관리자에서 설정
- 카테고리 : Suit 16px Regular
- 타이틀 : Suit 22px Bold
- 설명 텍스트 : Suit 16px Regular
  • 이미지 : 배경 색상 제거 후, 이미지 포함 640x480 사이즈로 추출(png)
* 이미지는 텍스트와 겹치지 않게 배치

2-1. 배경색상

  • 관리자에서 설정(배경색 색상코드)
