Text Color
텍스트 위계에 따라 Primary, Secondary, Tertiary 컬러로 사용하며 각 화면 구성 요소에 따라 이 외의 컬러를 사용합니다. Primitive 컬러는 브랜드를 대표하는 색상, 컴포넌트를 구성하는 색상으로 구성되어 있습니다.

구성요소 예시

Color/text/Primitive

브랜드 색상, 컴포넌트를 구성하는 색상입니다.
토큰 설명 참조값 Hex 컬러
Primary 브랜드 주 색상 (예시 : 버튼, 입력 폼, 강조 타이틀) brand/600 #FF8120
primary-press 브랜드 주 색상 (Press, Select) brand/800 #EE6002
secondary 컴포넌트 내 기본 블랙 색상 neutral/800 #333333
secondary-press 컴포넌트 내 기본 블랙 색상(Press, Select) neutral/900 #1A1A1A
selected 브랜드의 주 색상 중에서 선택된 상태를 강조할 때 사용하는 색상 brand/800 #EE6002
inverse 진한 배경에 사용하는 텍스트 색상 Base/white #FFFFFF

Color/text

전체 화면에서 자유롭게 사용하는 색상입니다.
토큰 설명 참조값 Hex 컬러
Primary 제목(Title,Sub Title) 및 본문(Body) 텍스트 색상 neutral/800 #1A1A1A
secondary 본문의 하위 위계인 서브 본문(Small) 텍스트 색상 neutral/600 #616161
teritiary 설명 텍스트 색상 neutral/500 #999999
brand-sub 브랜드 색상 이외의 강조 색상 Darkblue/900 #10185C
info 뱃지, 정보 등 신뢰 색상 Infomation/600 #1477F8
info-bold 뱃지, 정보 등 신뢰 색상(Select) Infomation/800 #1E40AF
success 완료 및 성공 색상 success/600 #43A047
success-bold 완료 및 성공 색상(Select) success/800 #166534
danger 주의 및 에러 색상 danger/600 #E53935
danger-bold 주의 및 에러 색상(Select) danger/800 #C62828
disabled 비활성화 색상 neutral/400 #BDBDBD
