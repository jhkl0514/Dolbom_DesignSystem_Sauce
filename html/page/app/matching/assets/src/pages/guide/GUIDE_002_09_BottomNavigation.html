Bottom Navigation
바텀 내비게이션(bottomNavigation)은 서비스의 주요 화면 간 이동을 돕는 컴포넌트입니다.

Anatomy, 컴포넌트 내부요소

  • 1. Container
  • 2. Text Label
  • 3. Icon

State, 상태

바텀 내비게이션의 아이콘은 Unselected, Selected 로 분류됩니다. 사용자 위치를 나타내므로 시각 일관성 고려를 위해 Selected Icon Color는 Neutral-900, Unselected Icon Color는 Neutral-600을 활용합니다.

Spec, 제작가이드

각 아이템의 너비 값은 디바이스 넓이를 아이템 개수로 1/n하여 계산합니다. 높이 값은 60px로 고정합니다.
