Chips
칩(Chips)은 사용자에게 제공해야 하는 입력 데이터, 필터 옵션, 기능을 상황에 따라 가변적으로 제공하여 작업 수행을 돕는 컴포넌트입니다.

Anatomy, 컴포넌트 내부요소

  • 1. Container
  • 2. Leading Icon(옵션)
  • 3. Text Label
  • 4. Trailing Icon(옵션)

Type, 종류

Option Chips는 칩을 선택해도 페이지가 이동하지 않고, 선택한 값은 즉시 시각적으로 반영(Active)되지만, 실제 저장은 ‘완료’ 버튼을 눌러야 적용됩니다.
Filter Chips는 칩을 선택하면 바로 필터가 적용되어 화면 데이터가 즉시 바뀌는 방식입니다.
