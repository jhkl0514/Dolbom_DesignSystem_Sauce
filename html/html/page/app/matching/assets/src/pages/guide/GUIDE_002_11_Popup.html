Popup
Popup(팝업)은 현재 페이지 위에 새로운 창을 띄우는 것으로, 새로운 창이 열려 기존에 보던 화면을 가리므로 사용자에게 추가적인 정보를 전달하거나 중요한 결정을 요구하는 컴포넌트입니다.
토스트 팝업(Toast Popup)은 일시적으로 사용자에게 전달해야 하는 간단한 메시지나 피드백을 제공하는 컴포넌트입니다.

  • 1. Container
  • 2. Title
  • 3. Text
  • 4. Button

Default Popup
Toast Popup

Variant, 형태

Default Popup_버튼1개
Default Popup_버튼2개
Toast Popup_한줄
Toast Popup_두줄

Spec, 제작가이드

팝업의 너비는 디바이스 너비의 80%로 보이도록 합니다. Scrim area(오버레이)색상은 #000000 60%로 지정합니다.
