Background Color
배경 색상에만 사용하는 컬러입니다. 텍스트, 배경, 아이콘 색상과 중복되는 색상이 있어도 지정된 토큰 색상으로만 사용합니다.

구성요소 예시

Color/bg/Primitive

브랜드 색상, 컴포넌트를 구성하는 색상입니다.
토큰 설명 참조값 Hex 컬러
Primary 브랜드 주 색상 (예시 : 버튼, 입력 폼) brand/600 #FF8120
primary-press 브랜드 주 색상 (Press, Select) brand/800 #EE6002
secondary 컴포넌트 내 기본 블랙 색상 neutral/200 #EEEEEE
secondary-press 컴포넌트 내 기본 블랙 색상(Press, Select) neutral/300 #E0E0E0
selected 선택되었을 때 배경 색상(예시 : 버튼, 입력 폼) brand/50 #FFF5E6
secondary-press 선택되었을 때 배경 색상(Press, Select) brand/200 #FFC77D

Color/bg

전체 화면에서 자유롭게 사용하는 색상입니다.
토큰 설명 참조값 Hex 컬러
Primary 기본 배경 색상 Base/whie #FFFFFF
secondary 두번째 기본 배경 색상 neutral/50 #FAFAFA
teritiary 목록과 유사한 구성 항목의 구분에 대한 배경 색상
(예시 : 푸터, 카드UI 내 배경) 		neutral/100 #616161
point-bold 브랜드 색상 중 연한 배경 색상 brand/100 #FFECB3
point-subtle 브랜드 색상 중 가장 연한 배경 색상 brand/50 #FFF5E6
brand-sub-bold 브랜드 색상 외 강조 하는 배경 강조 색상 brandSub/900 #10185C
brand-sub-subtle 브랜드 색상 외 강조 하는 배경 색상 (예시: 배너색상) brandSub/50 #EFF1FD
info-bold 뱃지, 정보 등 신뢰 강조 색상 Infomation/700 #1D4ED8
info-subtle 뱃지, 정보 등 신뢰 색상 (예시: 배너색상) Infomation/50 #EFF6FF
success-bold 완료 및 성공 강조 색상 success/700 #15803D
success -subtle 완료 및 성공 색상(예시: 배너색상) success/50 #E8F5E9
danger-bold 주의 및 에러 강조 색상(Select) danger/800 #D32F2F
danger-subtle 주의 및 에러 색상(예시: 배너색상) danger/50 #FFEBEE
inverse-bold 기본 배경색상 반전색상 neutral/800 #333333
inverse-bolder 기본 배경색상 반전 강조 색상 (Press, Select) neutral/900 #1A1A1A
disabled 비활성화 색상 neutral/200 #EEEEEE
