Selection Controls
셀렉션 컨트롤(Selection Controls)은 사용자가 선택 옵션들을 제어할 수 있도록 돕는 컴포넌트입니다. 다중선택을 위한 체크 박스, 다수의 옵션 중 하나의 선택을 위한 라디오 버튼, 효과 전환을 위한 토글 스위치로 나눕니다.

Anatomy, 컴포넌트 내부요소

  • 1. Container
  • 2. Mark
  • 3. Track
  • 4. Thumb

Type, 종류

1.Checkbox
체크박스는 제공된 옵션 항목 중 한 개 이상 선택할 수 있을 때 사용합니다.
2.Radio button
라디오 버튼은 제공된 옵션 항목 중 한개만 선택할 수 있을 때 사용합니다.
3.Switch
스위치는 제공된 화면의 설정을 켜거나 끌 때 사용합니다.

State, 상태

1.Checkbox
Table of Figma Propertie, 피그마 속성
Property Values Default Value
State Default, Pressed, Disabled Default
Selected True, False False
2.Radio button
Table of Figma Propertie, 피그마 속성
Property Values Default Value

Selected True, False False
3.Switch

Spec, 제작가이드

체크박스, 라디오버튼 사이즈는 최소 20x20px, 스위치는 최소 36px를, 세로 사이즈는 최소 20px를 사용합니다.
