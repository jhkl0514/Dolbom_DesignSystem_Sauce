List
리스트(LIst)는 콘텐츠나 텍스트를 반복적으로 나열하는 컴포넌트입니다.

Anatomy, 컴포넌트 내부요소

  • 1. Container
  • 2. Title
  • 3. Category
  • 4. Trailing Icon
  • 5. Icon
  • 6. Sub Text

Type, 종류

1.List
관련 항목을 목록 형태로 제공하여 사용자가 필요한 정보를 빠르게 탐색할 수 있도록 구성합니다.
Default
Contents
2.List_DivideText
항목명과 값을 나란히 배치하여 구조적으로 정보를 정리하고, 사용자가 핵심 데이터를 쉽게 식별할 수 있도록 돕습니다.
3.List_Notice
이용 과정에서 반드시 확인이 필요한 유의사항 또는 안내 내용을 시각적으로 강조하여 전달합니다.
