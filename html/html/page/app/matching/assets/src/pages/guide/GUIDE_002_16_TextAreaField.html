TextAreaField
텍스트 영역(TextAreaField)은 사용자가 키보드로 글자, 숫자, 기호 등이 조합된 여러 줄의 텍스트를 입력하는 경우에 사용하는 요소입니다.

Spec, 제작가이드

입력 영역은 텍스트의 길이를 고려하여 적절한 크기로 제공합니다.
입력 영역의 높이는 사용자가 예상되는 텍스트를 가독성 있게 확인할 수 있도록 적절한 크기로 제공하며, 별도의 스크롤 동작 없이 자연스럽게 내용을 입력하고 확인할 수 있도록 합니다.(Max height : 220px, 5~8줄 정도)
