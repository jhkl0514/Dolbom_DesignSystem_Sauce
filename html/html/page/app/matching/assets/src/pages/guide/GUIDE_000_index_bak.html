Design System Guide
이로움돌봄
디자인시스템 가이드
협업자들이 일관된 기준으로 UI를 제작할 수 있도록 디자인시스템과 퍼블리싱 가이드를 웹으로 제공합니다.
전체 구성
파운데이션

디자인의 기본 요소를 구성하는 색상, 타이포그래피, 간격 등을 정의하는 가이드입니다.

컴포넌트

일관된 사용자 경험을 위해 자주 사용하는 UI 요소를 모아둔 라이브러리입니다.

앱 스크린

모바일 환경에서 사용되는 화면 구성과 UI 요소를 시각적으로 정리하여 제공합니다.

웹 스크린

웹 환경에서 사용되는 화면 구성과 UI 요소를 시각적으로 정리하여 제공합니다.

스타일가이드

브랜드 정체성을 유지하기 위한 스타일 규칙과 활용 방안을 제공합니다.
